# Journal of Kinesiology and Wellness (JKW) Documentation

This repository contains the official JKW guide website.
It includes submission requirements, formatting standards, manuscript category guidance, templates, and author resources.

## Live Site

- Main guide website: https://wskwellness.github.io/jkw-guide/
- Submission portal: https://jkw.wskw.org/index.php/jkw/about/submissions
